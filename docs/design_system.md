# [Design System] 썬텍: 미래지향적 테크 프리미엄 (Tech Premium)

## 1. 브랜드 아이덴티티
- **키워드:** 미래(Future), 신뢰(Trust), 혁신(Breakthrough), 견고함(Solid).
- **컨셉:** 자동차 부품의 금속적 질감과 암흑 속의 빛나는 헤드라이트를 형상화한 다크 모드 지향 디자인.

## 2. 컬러 팔레트 (Role-based Colors)

| 역할 | HEX | 비고 |
| :--- | :--- | :--- |
| **Primary (Tech Blue)** | `#0A84FF` | 혁신과 첨단 기술을 상징하는 전기차 느낌의 블루 |
| **Surface (Background)** | `#050505` | 깊이 있는 블랙, 프리미엄 다크 모드의 기반 |
| **Secondary (Solid Silver)** | `#A1A1A6` | 부품의 금속 물질을 연상시키는 정제된 실버 |
| **Accent (Breaking Red)** | `#FF3B30` | 고정관념을 깨는 강렬한 포인트 (에러 및 강조) |
| **Text Primary** | `#FFFFFF` | 극강의 대비를 통한 가독성 확보 |
| **Text Secondary** | `#86868B` | 부차적 정보 전달을 위한 뉴트럴 그레이 |

## 3. 타이포그래피 (Typography Scale)
- **Primary Font:** `Inter` 또는 `Outfit` (구글 폰트) - 현대적이고 기계적인 정밀함 전달.
- **Display Head 1:** 4.5rem / 700 weight (메인 슬로건용)
- **Head 2:** 2.5rem / 600 weight (섹션 제목용)
- **Body Large:** 1.25rem / 400 weight (전문적인 설명용)
- **Caption:** 0.875rem / 500 weight (도면 수치 및 메타데이터용)

## 4. UI 컴포넌트 가이드

### 4.1 버튼 (Buttons)
- **Primary:** `Tech Blue` 배경, `White` 텍스트, 반경 8px의 마이크로 라운딩.
- **Secondary (Outline):** 선명한 실버 테두리, 호버 시 배경 채워짐.
- **Interaction:** 마우스 호버 시 미세한 글로우(Glow) 효과 적용.

### 4.2 스펙 카드 (Spec Cards)
- **Style:** Glassmorphism 효과 적용 (반투명 서피스 + 블러 효과).
- **Layout:** 도면 이미지는 왼쪽, 텍스트 데이터는 오른쪽 배치 (사무실 PC 최적화).

### 4.3 입력 필드 (Input Fields)
- **Focus State:** `Tech Blue` 테두리와 은은한 후광 효과로 사용자의 시선 유도.

## 5. 애니메이션 원칙
- **Entrance:** 요소들이 아래에서 위로 은은하게 나타나는 Fade-in-up 효과.
- **Micro-interaction:** 카탈로그 탐색 시 매끄러운 트랜지션 필터링.
- **Impact:** 메인 비주얼에서 자동차 부품의 작동 원리를 보여주는 시네마틱 루프 영상 활용.
