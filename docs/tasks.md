# [TASKS] 썬텍 홈페이지 개선 AI 개발 프롬프트 설계서

## 1. 개요
본 문서는 AI 코딩 파트너가 썬텍 홈페이지 개선 프로젝트를 단계별로 완수할 수 있도록 설계된 마이크로 태스크 모음입니다. 각 단계는 이전 단계의 결과물에 의존하며, PRD/TRD/시스템 가이드를 엄격히 준수해야 합니다.

---

### M0: 프로젝트 초기화 및 기술 스택 설정
**목표:** 고성능 Next.js 환경 및 백엔드 인프라 구축.

- [ ] **[TASK-0.1] Next.js 14+ 및 배포 환경 구성**
    - **컨텍스트:** 글로벌 런칭을 위한 고성능 웹 환경이 필요합니다.
    - **지시사항:** `Next.js` 14 이상의 App Router 구조로 프로젝트를 초기화하고, `Vercel` 호스팅 설정을 완료하세요.
    - **레퍼런스:** [TRD](file:///prd.md) 섹션 2.1, [TRD](file:///trd.md) 섹션 2.4.
    - **인수기준:** 무중단 배포 파이프라인(CI/CD) 확보 및 Hello World 페이지 표시.

- [ ] **[TASK-0.2] Supabase 백엔드 연동 및 보안 설정**
    - **컨텍스트:** 인증 및 데이터 저장을 위한 Backend-as-a-Service 구축.
    - **지시사항:** `Supabase` 프로젝트를 생성하고 로깅 및 RLS(Row Level Security) 원칙을 적용하여 DB를 연결하세요.
    - **인수기준:** 클라이언트 측에서 Supabase 인스턴스 연결 성공.

---

### M1: 핵심 UI 및 다크 모드 디자인 시스템 구축
**목표:** 테슬라 스타일의 화려하고 정밀한 인터페이스 구현.

- [ ] **[TASK-1.1] 글로벌 테마 및 스타일 가이드 적용**
    - **컨텍스트:** 첨단 기술력을 상징하는 다크 모드 기반 스타일 구현.
    - **지시사항:** [Design System](file:///design_system.md)을 바탕으로 컬러 토큰, 타이포그래피, 마이크로 애니메이션 기초를 `global.css` 및 변수로 정의하세요.
    - **인수기준:** 배경색 `#050505`, 텍스트 `#FFFFFF` 기반의 기본 레이아웃 완성.

- [ ] **[TASK-1.2] 고품질 내비게이션 및 다국어 초기화**
    - **컨텍스트:** 글로벌 연구원들을 위한 매끄러운 언어 전환 환경.
    - **지시사항:** `i18next`를 사용하여 KR/EN/CN 전환이 가능한 상단 내비게이션을 구현하세요.
    - **인수기준:** 클릭 한 번으로 헤더 메뉴의 언어가 즉시 바뀌는 기능 확인.

---

### M2: 핵심 기능 개발 (MVP: 인터랙티브 제품 카탈로그)
**목표:** PDF 다운로드의 불편함을 해소하는 정밀 웹 뷰어 구현.

- [ ] **[TASK-2.1] 제품 DB 스키마 구현 및 데이터 주입**
    - **컨텍스트:** [Database Design](file:///database_design.md) 기반의 기초 데이터 구성.
    - **지시사항:** 제품, 스펙, 미디어 테이블을 생성하고 썬텍의 샘플 데이터를 삽입하세요.
    - **인수기준:** SQL 질의를 통해 제품별 다중 스펙 조회가 가능할 것.

- [ ] **[TASK-2.2] 필터링 기능이 포함된 카탈로그 목록 뷰**
    - **컨텍스트:** 연구원이 원하는 부품을 즉시 찾을 수 있는 검색 환경.
    - **지시사항:** Next.js Server Components를 활용하여 고속 필터링이 가능한 카드 형태의 카탈로그를 구현하세요.
    - **인수기준:** 스펙 조건 변경 시 0.5초 이내에 목록이 갱신될 것.

---

### M3: 가상 공장 투어 및 기술 신뢰 섹션
**목표:** 썬텍의 제조 경쟁력을 시각적으로 증명.

- [ ] **[TASK-3.1] 시네마틱 제조 설비 갤러리**
    - **컨텍스트:** 생산 능력을 직관적으로 보여주는 영상 중심 섹션.
    - **지시사항:** [FEAT-2](file:///prd.md)를 구현하세요. Supabase Storage에 호스팅된 고품질 영상을 스트리밍하는 세련된 갤러리 뷰를 만드세요.
    - **인수기준:** 자동화 설비의 작동 모습이 PC 환경에서 부드럽게 재생될 것.

---

### M4: 전환 최적화 (문의 폼 및 마무리)
**목표:** 방문 고객을 실제 비즈니스 리드로 전환.

- [ ] **[TASK-4.1] 기술 문의 및 샘플 요청 관리 시스템**
    - **컨텍스트:** 1인 운영 환경에서도 안전하게 고객 문의를 수집.
    - **지시사항:** 유효성 검사가 적용된 Contact Form을 만들고, 제출된 내역이 Supabase DB에 저장되며 운영자에게 알림이 가도록 설계하세요.
    - **인수기준:** 문의 완료 후 감사 페이지가 노출되며 DB 상에 즉시 데이터가 기록될 것.
